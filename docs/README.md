# HybridOS

HybridOS — модульный над-слой, совмещающий гибридный GUI, 
сценарную автоматизацию на Lua и систему модульной загрузки.

## Key Features
- Modular subsystems
- Hybrid GUI (native + composited)
- Lua scripting runtime
- Bootloader-based dependency initialization
- Dual-licensed (MIT + Apache-2)

## Status
Early experimental prototype (v0.2.0)
